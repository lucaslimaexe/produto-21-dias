# **App Name**: Manifestation Potential Analyzer

## Core Features:

- Progress Indicator: Display an animated progress bar to show the current status of the analysis. The animation consists of both changing colors in a filled bar, and a numerical percentage.
- Analysis Steps Display: Cycle through the steps of analysis. Use each analysis step to compose a diagnostic status, such as 'Detecting your mental patterns'.
- Animated Status Icons: Include a sequence of graphical icons and descriptive messages related to identifying user patterns, calculating the user's potential, and preparing their transformation. The icons must be animated to give life to the process.
- Timed transitions: Implement timed transitions. This makes the display sequence more dynamic. Increment a timer regularly to smoothly update and move to the next analysis phase. 

## Style Guidelines:

- Primary color: Saturated purple (#9B51E0) for its association with spirituality and transformation.
- Background color: Very dark desaturated purple (#14111A).
- Accent color: Yellow-gold (#D4AF37) for contrast and to symbolize the revealing of inner potential.
- Headline font: 'Poppins' sans-serif for the diagnostic steps.
- Body font: 'Inter' sans-serif for status messages. Using Inter with Poppins because longer text may be used.
- Use smooth transitions and subtle animations for the progress bar and icons to enhance the user experience. A continuous gentle motion gives life to the display.
- Center all elements on the screen with a dark gradient background to focus the user's attention on the diagnostic process.
